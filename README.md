# Not Your Average Agency - Website

A premium, high-performance web design agency website built with React, Vite, and TypeScript.

## Features

-   **Tech Stack**: React 19, Vite, TypeScript, Tailwind CSS (via CDN), Framer Motion.
-   **Design**: Modern, "Cyberpunk Luxury" aesthetic with glassmorphism and smooth animations.
-   **Performance**: Optimized build with strict type checking.
-   **SEO**: Configured with proper metadata and tags.

## getting Started

### Prerequisites

-   Node.js (v18 or higher recommended)
-   npm or yarn

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/navneethbekalbiz/NotYourAverage-Web-Design-Agency-Website-.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Set up environment variables:
    -   Copy `.env.example` to `.env`:
        ```bash
        cp .env.example .env
        ```
    -   Update `.env` with your API keys (e.g., `GEMINI_API_KEY`).

### Development

Run the development server:

```bash
npm run dev
```

### Build

Build for production:

```bash
npm run build
```

This commands runs `tsc` for type checking followed by `vite build`.

## Deployment

This project is ready to be deployed on Vercel.

1.  Push your code to GitHub.
2.  Import the project in Vercel.
3.  Vercel will automatically detect Vite and set the build settings.
4.  Add your Environment Variables in the Vercel project settings.
